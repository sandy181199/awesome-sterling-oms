# Contributing

PRs welcome. A few ground rules to keep this list useful:

- One entry per PR, ideally. Makes review easier.
- Entries need a real, working link. No dead links, no placeholder URLs.
- Keep descriptions short — a single line, no marketing copy.
- New sections need a handful of entries to justify existing, not just one.
- Don't add your own product/blog unless it's genuinely useful to someone working with Sterling OMS, not just self-promotion.
- Run through the [awesome-lint](https://github.com/sindresorhus/awesome-lint) rules before opening a PR if you can — alphabetical order within sections, consistent formatting, etc.

If you're fixing a `TODO: add link` placeholder, just replace the line — no need to touch anything else nearby.
